
# API de Cursos com Flask

## 📌 Descrição
API simples para gerenciamento de cursos utilizando Flask e JSON como banco de dados.

## 🚀 Como executar

1. Instale o Flask:
---
pip install flask

2. Execute o projeto:
---
python app.py

3. Acesse:
---
http://127.0.0.1:5000

## 📌 Endpoints

- POST /cursos
- GET /cursos
- GET /cursos/{id}
- PUT /cursos/{id}
- DELETE /cursos/{id}

## 🧪 Testes

Recomenda-se usar o Postman ou Insomnia.
